    技术指标的数学原理

    记号

    • p p p: price, 价格
    • V V V: volume, 交易量
    • σ \sigma σ: standard deviation, 标准差
    • E E E: extreme point, 极值
    • 均值
      • μS \mu_S μS​: simple moving average, 简单移动平均线
      • μE \mu_E μE​: exponential moving average, 指数移动平均线
      • μD \mu_D μD​: double exponential moving average, 双指数移动平均线
      • μT \mu_T μT​: triple exponential moving average, 三指数移动平均线
      • μG \mu_G μG​: generalized DEMA, 广义双指数移动平均线

    数学基础

    • Mean Deviation: MD(p)=1N∑∣pi−μ(p)∣ \text{MD}(p) = {1\over N} \sum | p_i - \mu(p) | MD(p)=N1​∑∣pi​−μ(p)∣

    网红指标

    [EMA] Exponential Moving Average 指数移动平均线

    • μEn=rpn+(1−r)μEn−1 {\mu_E}_n = r p_n + (1-r) {\mu_E}_{n-1} μE​n​=rpn​+(1−r)μE​n−1​

    [MACD] Moving Average Convergence / Divergence 指数平滑移动平均线

    • m=μE,r1−μE,r2 m = \mu_{E, r_1} - \mu_{E, r_2} m=μE,r1​​−μE,r2​​

    [KDJ] Stochastic Oscillator 随机指标

    • K=pC−min⁡(pL)max⁡(pH)−min⁡(pL)⋅100 K = { p_C - \min(p_L) \over \max(p_H) - \min(p_L) } \cdot 100 K=max(pH​)−min(pL​)pC​−min(pL​)​⋅100
    • D=μ(K) D = \mu(K) D=μ(K)
    • J=rK+(1−r)D J = r K + (1-r) D J=rK+(1−r)D

    [BOLL] Bollinger Bands 布林带

    • pˉ=pH+pL+pC3 \bar{p} = { p_H + p_L + p_C \over 3 } pˉ​=3pH​+pL​+pC​​, 称作 Typical price 典型价格
    • bM=μ(pˉ) b_M = \mu( \bar{p} ) bM​=μ(pˉ​)
    • bU/L=bM±Nσ(pˉ) b_{U/L} = b_M \pm N \sigma( \bar{p} ) bU/L​=bM​±Nσ(pˉ​)

    [RSI] Relative Strength Index 相对强度指数

    dp = close - close[-1]
up   = abs(dp) if dp > 0 else 0
down = abs(dp) if dp < 0 else 0
upAvg   = r * upAvg[-1]   + (1-r) * up
downAvg = r * downAvg[-1] + (1-r) * down
rsi = upAvg / (upAvg + downAvg) * 100

    [CCI] Commodity Channel Index 商品通道指数

    • pˉ=pHT+pLT+pC3 \bar{p} = { p_{HT} + p_{LT} + p_C \over 3 } pˉ​=3pHT​+pLT​+pC​​
    • pHT p_{HT} pHT​: T T T 时间内最高价
    • c=pˉ−μS(pˉ)1.5  MD(pˉ)⋅100 c = { \bar{p} - \mu_S(\bar{p}) \over 1.5 \; \text{MD}(\bar{p}) } \cdot 100 c=1.5MD(pˉ​)pˉ​−μS​(pˉ​)​⋅100

    Williams' R 威廉姆斯的R

    • max⁡(pH)−pCmax⁡(pH)−min⁡(pL)⋅100 { \max(p_H) - p_C \over \max(p_H) - \min(p_L) } \cdot 100 max(pH​)−min(pL​)max(pH​)−pC​​⋅100

    [SAR] Parabolic Stop And Reverse 抛物线反转

    • 如果在涨(收 > 开): E=max⁡(p) E = \max(p) E=max(p)
    • 如果在跌(收 < 开): E=min⁡(p) E = \min(p) E=min(p)
    • sn=sn−1+a(En−1−sn−1) s_n = s_{n-1} + a ( E_{n-1} - s_{n-1} ) sn​=sn−1​+a(En−1​−sn−1​)
    • a a a: 加速因子

    [Aroon] (dawn's early light) 阿隆 (黎明的曙光)

    • AH/L=T−TH/LT⋅100 A_{H/L} = {T - T_{H/L} \over T} \cdot 100 AH/L​=TT−TH/L​​⋅100
    • TH/L T_{H/L} TH/L​: 在 T T T 时间内, 从最高/最低价算起经过的时间
    • 示例: 如果当前最低点是 T T T 内的最低点, 说明价格在下跌, 则 TL=0,AL=100 T_L = 0, A_L = 100 TL​=0,AL​=100

    • c=∑Δp∑∣Δp∣⋅100 c = { \sum \Delta p \over \sum |\Delta p| } \cdot 100 c=∑∣Δp∣∑Δp​⋅100

    [DMI] Directional Movement Index 方向运动指数

    dH = high[-1] - high
dL = low - low[-1]
if (dH < 0 and dL < 0):
    plusDM = 0
    minusDM = 0
plusDM = dH if dH > dL else 0
minusDM = 0 if dH > dL else dL 
trueHigh = max( high[0], close[-1] )
trueLow  = min( low[0] , close[-1] )
trueRange = trueHigh - trueLow

sumPlusDM    = sumPlusDM[-1]    * (1-1/T) + plusDM
sumMinusDM   = sumMinusDM[-1]   * (1-1/T) + minusDM
sumTrueRange = sumTrueRange[-1] * (1-1/T) + trueRange

plusDI  = sumPlusDM  / sumTrueRange * 100
minusDI = sumMinusDM / sumTrueRange * 100

dmi = (plusDI - minusDI) / (plusDI + minusDI)
