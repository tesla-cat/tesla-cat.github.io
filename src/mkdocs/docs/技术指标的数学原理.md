
### 记号

- $` p `$: price, 价格
- $` V `$: volume, 交易量
- $` \sigma `$: standard deviation, 标准差
- $` E `$: extreme point, 极值
- 均值
    - $` \mu_S `$: simple moving average, 简单移动平均线
    - $` \mu_E `$: exponential moving average, 指数移动平均线
    - $` \mu_D `$: double exponential moving average, 双指数移动平均线
    - $` \mu_T `$: triple exponential moving average, 三指数移动平均线
    - $` \mu_G `$: generalized DEMA, 广义双指数移动平均线

### 数学基础

- Mean Deviation: $` \text{MD}(p) = {1\over N} \sum | p_i - \mu(p) | `$

## 网红指标

### **[EMA]** Exponential Moving Average 指数移动平均线

- $` {\mu_E}_n = r p_n + (1-r) {\mu_E}_{n-1} `$

### **[MACD]** Moving Average Convergence / Divergence 指数平滑移动平均线

- $` m = \mu_{E, r_1} - \mu_{E, r_2} `$

### **[KDJ]** Stochastic Oscillator 随机指标

- $` K = { p_C - \min(p_L) \over \max(p_H) - \min(p_L) } \cdot 100 `$
- $` D = \mu(K) `$
- $` J = r K + (1-r) D `$

### **[BOLL]** Bollinger Bands 布林带

- $` \bar{p} = { p_H + p_L + p_C \over 3 } `$, 称作 Typical price 典型价格
- $` b_M = \mu( \bar{p} ) `$
- $` b_{U/L} = b_M \pm N \sigma( \bar{p} ) `$

### **[RSI]** Relative Strength Index 相对强度指数

```python
dC = close - close[-1]
up   = abs(dC) if dC > 0 else 0
down = abs(dC) if dC < 0 else 0
upAvg   = r * upAvg[-1]   + (1-r) * up
downAvg = r * downAvg[-1] + (1-r) * down
rsi = upAvg / (upAvg + downAvg) * 100
```

### **[CCI]** Commodity Channel Index 商品通道指数

- $` \bar{p} = { p_{HT} + p_{LT} + p_C \over 3 } `$
- $` p_{HT} `$: $` T `$ 时间内最高价
- $` c = { \bar{p} - \mu_S(\bar{p}) \over 1.5 \; \text{MD}(\bar{p}) } \cdot 100 `$

### Williams' R 威廉姆斯的R

- $` { \max(p_H) - p_C \over \max(p_H) - \min(p_L) } \cdot 100 `$

### **[SAR]** Parabolic Stop And Reverse 抛物线反转

- 如果在涨(收 > 开): $` E = \max(p) `$
- 如果在跌(收 < 开): $` E = \min(p) `$
- $` s_n = s_{n-1} + a ( E_{n-1} - s_{n-1} ) `$
- $` a `$: 加速因子

## 考察均值的指标

### **[DEMA]** Double Exponential Moving Average 双指数移动平均线

- 延迟小于 $` \mu_E `$, 原理即下式中移除双延迟项
- $` \mu_D(p) = 2 \mu_E(p) - \mu_E(\mu_E(p)) `$

### **[TEMA]** Triple Exponential Moving Average 三指数移动平均线

- $` \mu_T(p) = 3 \mu_E(p) - 3 \mu_E(\mu_E(p)) + \mu_E(\mu_E(\mu_E(p))) `$

### Generalized DEMA 广义双指数移动平均线

- $` \mu_G(p) = (1+r) \mu_E(p) - r \mu_E(\mu_E(p))  `$

### **[T3]** 三阶广义双指数移动平均线

- $` T3(p) = \mu_G(\mu_G(\mu_G(p))) `$

## 考察动量的指标

### **[Aroon]** (dawn's early light) 阿隆 (黎明的曙光)

- $` A_{H/L} = {T - T_{H/L} \over T} \cdot 100 `$
- $` T_{H/L} `$: 在 $` T `$ 时间内, 从最高/最低价算起经过的时间
- 示例: 如果当前最低点是 $` T `$ 内的最低点, 说明价格在下跌, 则 $` T_L = 0, A_L = 100  `$

### **[CMO]** Chande Momentum Oscillator 钱德动量振荡器

- $` c = { \sum \Delta p \over \sum |\Delta p| } \cdot 100 `$

### **[DMI]** Directional Movement Index 方向运动指数

```python
dH = high[-1] - high
dL = low - low[-1]
if (dH < 0 and dL < 0):
    plusDM = 0
    minusDM = 0
plusDM = dH if dH > dL else 0
minusDM = 0 if dH > dL else dL 
trueHigh = max( high[0], close[-1] )
trueLow  = min( low[0] , close[-1] )
trueRange = trueHigh - trueLow

sumPlusDM    = sumPlusDM[-1]    * (1-1/T) + plusDM
sumMinusDM   = sumMinusDM[-1]   * (1-1/T) + minusDM
sumTrueRange = sumTrueRange[-1] * (1-1/T) + trueRange

plusDI  = sumPlusDM  / sumTrueRange * 100
minusDI = sumMinusDM / sumTrueRange * 100

dmi = (plusDI - minusDI) / (plusDI + minusDI)
```

### **[ATR]** Average True Range 平均真实范围

- trueRange 同上
- $` \text{ATR} = \mu(\text{trueRange}) `$

### Ultimate Oscillator 终极摆动指标

- trueLow, trueRange 同上
- $` a_i = T_i \mu_{S, T_i}(p_C - \text{trueLow}) `$
- $` b_i = T_i \mu_{S, T_i}(\text{trueRange}) `$
- $` u = \sum r_i {a_i \over b_i} `$

### Momentum 动量

- $` m_n = p_n - p_{n-T} `$

### My Momentum and Force 我自创的动量和力

- 物理原理: $` p = mv, F = d_t p = v d_t m + m d_t v `$
- 换成交易: $` m = V d_t p, F = d_t p \; d_t V + V \; d^2_t p `$

### **[TRIX]** Rate-Of-Change of a Triple Smooth EMA 三重平滑EMA的变化率

- $` m = \mu_E(\mu_E(\mu_E(p))) `$
- $` \text{trix} = { m_n - m_{n-1} \over m_n } \cdot 100 `$

## 考察周期的指标

### Hilbert Transform and Analytic Signal 希尔伯特变换与解析信号

!!! Note
    希尔伯特变换也被用于脑电 EEG 数据分析!

- 参见 [`scipy.signal.hilbert`](https://docs.scipy.org/doc/scipy/reference/generated/scipy.signal.hilbert.html)

```python
def hilbertAnalysis(y, t, df = None):
    sampleFreq = len(t) / (t[-1] - t[0])
    if df is None: df = pd.DataFrame()
    y = hilbert(y)
    df['htReal'] = np.real(y)
    df['htImag'] = np.imag(y)
    df['htAbs'] = np.abs(y)
    df['htPhase'] = np.unwrap(np.angle(y))
    df['htFreq'] = sampleFreq * np.gradient(df['htPhase']) / (2*np.pi)
    return df
```
